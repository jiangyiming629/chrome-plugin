# JSON Formatter Chrome Extension

A lightweight Chrome extension to validate, format, and minify JSON.

## Features

- Format JSON with 2-space indentation
- Minify JSON
- Validate JSON and show readable parse errors
- Copy result to clipboard
- Auto-save input in extension local storage

## Load in Chrome

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this folder: `json-formatter-extension`

## Usage

- Click the extension icon
- Paste JSON
- Click **Format** or **Minify**
- Optional: `Cmd/Ctrl + Enter` to format quickly
